# RNN-sentiment-analysis-for-IMDB
Sentiment analysis for IMDB dataset using RNN


In this project, we are doing the following steps:

# Convert to lowercase
# Remove URLS
# Remove HTML tags
# Remove punctuations
# Remove strop word
# Perform stemming 
# Encode Sentiments
# Perform vectorization (TF- IDF)
# Train & Testmodel
# Build the RNN model
# Training the RNN
# Evaluate the Model
